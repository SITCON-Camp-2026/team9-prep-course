# notes 資料夾

這裡放你寫給 Coding Agent 看的自然語言草稿。

請從 `profile-note.template.md` 複製一份，改成：

```text
notes/<你的 GitHub 帳號>.md
```

例如：

```text
notes/octocat.md
```
